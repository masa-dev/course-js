## 概要

点（車）に既定のコースを壁に当たらないように走らせる

## 操作方法

### パソコン

- 右回転

    「a」または「→」

- 左回転

    「d」または「←」

- 画面の更新

    「Enter」または「Space」

### スマートフォン

各種ボタンで「右回転」、「左回転」、「画面の更新」ができる
